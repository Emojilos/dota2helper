# Живой гейт TASK-008 — оверлей поверх реальной Dota 2

Прогон на Windows-машине владельца, 2026-07-19. Dota 2 в режиме **borderless**
(«окно без рамки») — целевой режим PRD; поверх эксклюзивного фулскрина
always-on-top-окна не отрисовываются (ограничение ОС, подтверждено вживую).

## Результаты по шагам гейта

| Шаг | Результат |

| 1. Оверлей виден поверх игры (тестовый placeholder) | OK — без альтаба, borderless |

| 4. FPS с оверлеем и без | **120 / 120 — просадка 0%** (критерий ≤5%), счётчик FPS в одинаковой сцене |

## Находка гейта: RegisterHotKey не работает поверх Dota

`electron.globalShortcut` (RegisterHotKey) срабатывал в любом приложении,
кроме сфокусированной Dota — raw-input игры перехватывают ввод до системного
матчинга хоткеев (electron#27240). Починено win32-бэкендом на `uiohook-napi`
(низкоуровневый observe-only хук, событие не подавляется, инъекций в процесс
игры нет — INV3): коммит `01ae8b9`, живое подтверждение владельцем в тот же
день. Детали дизайна — doc-комментарии `src/main/hotkeys/`.

## Условия замера

- Оверлей: базовый инстанс `OverlayWindow` 320×120, transparent,
  always-on-top `'screen-saver'`, placeholder-контент (data:-URL).
- Замер FPS: одинаковая сцена (база), счётчик кадров, приложение
  запущено (`npm run dev`) vs полностью закрыто.
- Машина владельца (игровая, Windows), Dota 2 текущего патча на 2026-07-19.
